Le problème min-max

Problème trouver le plus grand élément et le plus petit élément d’une collection ordonnée (e.g., liste, tableau) non-vide.

Solution étudiant-e en licence

def min_max_etudiant(arr):
    the_min = arr[0]
    the_max = arr[0]
    for v in arr:
        if v < the_min:
            the_min = v
        if v > the_max:
            the_max = v

    return the_min, the_max

Solution le trieur

def min_max_sorter(arr):
    s = sorted(arr)
    return s[0], s[-1]

Solution Pythonista

def min_max_pythonista(arr):
    return min(arr), max(arr)
